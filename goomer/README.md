## Installation

Clone the repository:

```shell
git clone https://github.com/JuanAlmeida12/goomer.git
```

Open prompt/bash into repository directory and run the following command:
```shell
npm install
```
or just:
```shell
npm i
```
Now you are able to run the server.

## Running

Open prompt/bash into repository directory and run the following command:
```shell
npm start
```
or with `yarn`:
```shell
yarn start
```
